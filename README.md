# Feel That Pain - Linktree Personalizado

### **Hub de Enlaces y Lanzamientos Musicales**
Este es un proyecto estático desarrollado con **HTML y CSS** diseñado para centralizar mi presencia digital como productor de música electrónica (**Feel That Pain**). A diferencia de un Linktree convencional, este sitio permite un control total sobre la estética, el rendimiento y la integración de contenido multimedia.

---

## Características
* **Diseño Responsivo:** Optimizado para dispositivos móviles, asegurando una navegación fluida desde Instagram, TikTok y otras redes sociales.
* **Identidad Visual:** Estética personalizada alineada con el concepto artístico de "Feel That Pain".
* **Acceso Directo:** Enlaces rápidos a plataformas de streaming (Spotify, Yt Music).
* **Lightweight:** Código limpio sin dependencias pesadas, garantizando una carga instantánea.
